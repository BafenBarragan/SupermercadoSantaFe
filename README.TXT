------------------------------------------------------------------------
Este repositorio contiene los archivos de Código fuente del proyecto del 
sistema de nómina para el supermercado Santa Fe, ejercicio propuesto en 
el ambiente de aprendizaje como parte de la actividad #1 de la guía #3 
del bloque de estudios de Construcción del Software.
------------------------------------------------------------------------

- PROJECT TITLE: Actividad #1

- PURPOSE OF PROJECT Desarrollar el proyecto de un sistema de nómina de un 
supermercado de la ciudad de Bogotá. 

- VERSION or DATE: 1.0

- HOW TO START THIS PROJECT: El proyecto contiene dos clase, la clase Empleado
y la clase SantaFe, la primera contiene todos los métodos para leer y obtener
la información de cada empleado. La clase SantaFe permite inicializar una 
interfaz de línea de comando para así ejecutar el programa.

- AUTHORS: Bafen Ricardo Barragán Sánchez.
